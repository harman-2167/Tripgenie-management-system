# Tripgenie-management-system
TripGenie – A travel booking and management application built using Python, Tkinter, and MySQL.
=====Features=====
- Hotel Booking System
- Transport Booking System
- Complaint & Feedback Module
- Admin Dashboard
- Region & Booking Management
- User-friendly GUI Interface

=====Tech Stack=====
- Python 
- Tkinter (GUI)
- MySQL (Database)
- SQL (Queries & Database Operations)


=====Database Setup=====
1. Open MySQL Workbench
2. Import `Trip_genie.sql` file
3. Run the database script
4. Update your MySQL credentials in the Python code:
